# Send Emails with Node/Express

## Tech/Libraries
- Node/Express
- Nodemailer
- Sendgrid
- Dotenv
- Express-Async-Errors
- Http-status-codes
- Nodemon

## Run

- clone this repo into desired directory
- Navigate to that directory and create new .env file with the following values:
    - PORT = 
    - SENDGRID_API_KEY =
- from your terminal, run:
```
npm install

#Run localhost:3333
npm run dev
```
